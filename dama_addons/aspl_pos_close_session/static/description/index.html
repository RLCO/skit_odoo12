Odoo 12 POS Close Session

Manage cash control while closing session.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Close session from POS.
  • While closing the session,It will manage
    • - Cash Control.
    - Print pdf Z report(end of the session report).
    - Print Z report from thermal printer connected via POSBox.
    - Send Z report via email to selected users.

Available for Enterprise

How to use

  • To enable generate pdf and send email of close session report here is the configuration.

  • To enable login with direct POS here is the configuration.

  • Cashier allows put some money as a open balance in cashdrawer.


  • Click POS Session close to open closing cash control popup.

  • Click Session close to close session and after print Z report.

  • Email will be sent to user.
