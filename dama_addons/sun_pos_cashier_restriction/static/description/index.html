Odoo POS Cashier Restriction

Key features of this module

  • Using this module you can hide the collection details for cashier.
  • Compatible for community and enterprise version of odoo

  • Here if user is not a standard cashier than check a Configuration.


  • Here if user is not a standard cashier than user can see the all the details of collection.


  • Here if user is standard cashier than check a Configuration.


  • Here if user is a standard cashier than user cannot see the all the details of collection.
